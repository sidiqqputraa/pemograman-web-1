PRAKTIKUM PEMOGRAMAN WEB

Praktikum Pemograman Web memberikan keterampilan kepada mahasiswa dalam pembuatan situs web yang dinamis dan dapat berintraksi dengan user. materi meliputi : Teknologi web, Pembuatan situs, HTML, CSS, PHP, MySQL dan Visual Studio Code.

Keterangan:

  1. Warna border adalah yellowgren
  2. Gunakan transformasi teks upercase
  3. Paragraf umum menggunakan underline-red untuk kalimat dengan garis bawah merah
  4. Buatlah class bernama shadow untuk memberikan efek bayangan pada teks
